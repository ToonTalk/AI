Quantum Duck Pond

We fire tiny 🦆 rubber ducks as a wave toward a wall with one or two slits. When the wave reaches the screen on the right, a single duck is detected at one spot. Repeat this many times to build up a distribution.

  1. Choose slits: Single → one broad band. Double → interference fringes. Which‑path tagging scrambles the relative phase between the two paths so fringes vanish (both slits stay open).
  2. Launch to emit a wave packet; Measure position (or enable Passive auto detector) to record a duck.
  3. Run N trials repeats launch→measure and shows a histogram of hits on the right.
  4. Speed changes how fast the wave evolves. Ducks always appear just after the wave arrives.
  5. Show phase toggles the color scheme to visualize the wave's phase (positive and negative parts) instead of only its intensity.

Detectors

  • Which‑path detector — Both slits stay open, but we add a tiny “tag” that lets you tell which slit the duck took. That destroys the relative phase between the two paths, so the interference fringes vanish. You’ll see two broad single‑slit envelopes instead of bright/dark stripes.
  • Passive auto detector — Acts like a screen that clicks on its own. When the wave reaches the detector band, it triggers a detection almost immediately (random in time, weighted by the local intensity). Turn this on if you don’t want to press Measure position each launch.
