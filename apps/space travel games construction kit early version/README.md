# Space Travel Games Construction Kit

This is a browser-based reconstruction inspired by the paper in `BBC Jam 11915355_25.pdf`, especially the described Space Games Construction Kit.

Open `index.html` in a browser to run it. The prototype includes:

- Phase-based rescue and Lunar Lander missions.
- Behaviour gadgets with sliders for velocity, rock throwing, gravity, fuel, air, gauges, and autopilot.
- Pseudo-Logo code boxes based on the fragments described in the paper.
- A small metagame panel with designer, scientist, programmer, and historian viewpoints.
- Live canvas playfield, keyboard controls, velocity arrow, fuel/speed/time readouts, and graphing gauges.

Controls:

- Rescue: arrow keys or WASD throw rocks in the opposite direction to create thrust.
- Lander: Up/WASD fires thrusters.
- Record captures a Lunar Lander control trace; Autopilot replays it after a reset.
