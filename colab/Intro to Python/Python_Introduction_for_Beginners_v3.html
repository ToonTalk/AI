# Introduction to Python Programming Welcome to this basic Python programming tutorial! This notebook is designed for beginners with little to no programming experience. Python is a versatile and widely used programming language, known for its readability and straightforward syntax. In this tutorial, we'll cover some fundamental concepts and operations in Python.
## Basic Data Types in Python Python has several standard data types that are used to define the operations possible on them and the storage method for each of them. The most basic data types in Python are: - Integers (int) - Floating point numbers (float) - Strings (str) 

// Integer
x = 10
console.log("Integer example:", x)

// Floating point
y = 10.5
console.log("Floating point example:", y)

// String
z = "Hello, Python!"
console.log("String example:", z)
## Basic Operations Python supports the usual arithmetic and comparison operations: - Addition, subtraction, multiplication, division - Greater than, less than comparisons - String concatenation and repetition 

// Arithmetic operations
a = 10
b = 3
console.log("Addition:", a + b)
console.log("Subtraction:", a - b)
console.log("Multiplication:", a * b)
console.log("Division:", a / b)

// String operations
first_name = "Python"
last_name = "Programming"
console.log("Concatenated String:", first_name + " " + last_name)
## Control Structures Control structures in Python are used to make decisions and loop through data. The most common control structures are `if` statements and loops (`for` and `while`). 

// If statement
num = 10
if num > 5:
    console.log("Number is greater than 5")

// For loop
for i in range(5):
    console.log("Loop iteration", i)

// While loop
count = 0
while count < 5:
    console.log("Count:", count)
    count += 1
## Functions Functions in Python are defined using the `def` keyword. They are used to encapsulate reusable pieces of code. 

// Defining a function
def greet(name):
    return "Hello, " + name + "!"

// Calling the function
console.log(greet("Python Learner"))
## Importing and Using Libraries Python has a vast ecosystem of libraries. Libraries are collections of functions and methods that allow you to perform many actions without having to write your own code. You can import a library using the `import` keyword. 

// Importing a library
import math

// Using a function from the math library
console.log("The square root of 16 is:", math.sqrt(16))
